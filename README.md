# 🎯 Num Secreto — Jogo de Adivinhação

> Projeto de jogo interativo desenvolvido com **Python + Streamlit**, publicado como aplicação web com deploy na nuvem via **Streamlit Community Cloud**.

---

## 🕹️ Sobre o Projeto

**Num Secreto** é um jogo de adivinhação onde o usuário tem **3 tentativas** para descobrir um número sorteado aleatoriamente entre **1 e 15**.

A cada tentativa, o jogo fornece uma dica indicando se o número informado é **maior** ou **menor** que o número secreto, tornando a experiência mais estratégica e divertida.

🔗 **Acesse o jogo ao vivo:** [adivinheonro.streamlit.app](https://adivinheonro.streamlit.app/)

---

## ✨ Funcionalidades

- 🎲 Sorteio aleatório de número a cada nova partida
- 🔢 Entrada do usuário para números de 1 a 15
- 💬 Feedback imediato após cada tentativa (maior / menor)
- ❤️ Contador de tentativas restantes (máximo: 3)
- 🏆 Mensagem de vitória ou derrota ao final da partida
- 🔄 Botão para reiniciar e jogar novamente

## 💡 Aprendizados e Conceitos Aplicados

Este projeto foi desenvolvido com foco em aprendizado prático e demonstra domínio dos seguintes conceitos:

- **Gerenciamento de estado** com `st.session_state` no Streamlit
- **Controle de fluxo** de jogo (tentativas, vitória, derrota)
- **Interatividade** com widgets do Streamlit (`st.number_input`, `st.button`)
- **Deploy de aplicação Python** na nuvem com Streamlit Community Cloud
- **Boas práticas** de estruturação de código e separação de lógica
